# 响应式规范

## 策略
- `>= 1024px ： PC UI`
- `< 1024px  ： Mobile UI`

## 适配分辨率清单
**PC端**
- 2160
- 1920
- 1536
- 1440
- 1366
- 1024

**Mobile端**
- 768
- 428
- 414
- 390
- 375
- 360

## UI设计规范
1.【0-1023】
默认等比缩放MB设计稿

2.【1024-1440】
默认等比缩放PC设计稿

3.【大于1440】
不等比缩放，版心(1376px)内容居中

## 数据支撑
官网主要站点 近3个月（2023.04.14-2023.07.12）各分辨率访问占比GA数据
