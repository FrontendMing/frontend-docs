## 使用方法
### 第一步：对 jackery-util 目录下，执行 npm init jackery 命令

```shell
$ cd jackery-util
$ npm init jackery
  Need to install the following packages:
  create-jackery
  Ok to proceed? (y)
```

第一次使用时，会出现是否安装附带包的提示，这些包是需要的，所以需要输入 y 并回车确认

### 第二步：进入 CLI 界面，选择要使用的功能
